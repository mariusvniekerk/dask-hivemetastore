Overview
--------

A set of bindings to allow interoperability between dask and the hive metastore

Installation / Usage
--------------------

To install use pip:

    $ pip install dask-hivemetastore


Or clone the repo:

    $ git clone https://github.com/mariusvniekerk/dask-hivemetastore.git
    $ python setup.py install
